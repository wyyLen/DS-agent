欢迎来到dsagent项目的原始仓库。**dsagent** 是一个针对数据科学任务设计的大模型智能体系统，致力于提升大语言模型在复杂数据科学工作中的推理与决策能力。当前大语言模型（LLM）虽然能够通过自然语言交互执行各类任务，但在面临数据依赖复杂、流程高度非线性和严苛任务约束的数据科学场景时，现有方法（如直接代码生成、ReAct、CoT等）常常出现步骤不完整、指令不准确、泛化或遗漏等问题。

为解决上述挑战，**dsagent** 提出了两大关键创新技术：

1. **基于工作流的检索增强规划**：构建并利用数据科学经验知识库，结合文本与工作流双重知识，动态增强智能体的任务分解与执行能力，实现了高效、闭环的知识驱动任务规划，显著提升了任务可靠性与泛化性。
2. **基于树搜索的自主探索机制**：面向复杂任务，实现对数据科学流程与探索空间的系统建模，通过高效的流程树搜索策略、反思机制和节点优化，显著提升了在广泛解空间下的自主探索和解决能力。

系统内置丰富的数据科学经验知识库，并支持三种智能体调用模式：规划执行、自主探索与动态调用。系统提供Web可视化交互界面，面向数据科学家与分析师实现一站式、自然语言的数据科学分析，助力高效、可靠的数据科学工作流自动化。
